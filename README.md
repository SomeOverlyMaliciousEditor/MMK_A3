# MMK_A3 Smarte Küche

Wir sollen eine App entwerfen, welche etablierte und cutting-edge IoT und smarte Küchengeräte 
berücksichtigt.
Dabei soll diese App sowohl eine zentrale Plattform darstellen mithilge von einer guten UI, sowie Sprachsteuerung, KI & mehr. Die App stellt eine zentrale Plattform für arbeiten in der Küche dar.

Wir haben uns auf folgende Parameter festgesetzt.
- Größe des Bediengerätes: Tablet
- Designsprache: Apple/iOS
- Sprachassistent: Siri
